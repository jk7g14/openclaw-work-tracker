# Work Tracker - 다우오피스 근무시간 추적

## 중요: 반드시 스크립트를 실행해야 한다

이 스킬은 사진을 분석한 후 **반드시** `scripts/calc_hours.py` 스크립트를 실행해서 계산해야 한다.
절대로 직접 시간을 계산하지 마라. 스크립트 실행 결과만 사용해라.

## 동작 순서

1. 사진에서 출퇴근 데이터 추출
2. JSON 형식으로 변환
3. **`python3 scripts/calc_hours.py --data '<JSON>'` 실행** ← 이 단계를 절대 건너뛰지 마라
4. 스크립트 출력 결과를 한국어로 포맷해서 응답

## 회사 규칙
- 주 40시간 근무 (월~금)
- 코어타임: 11:00 ~ 17:00
- 점심시간: 12:30 ~ 13:30 (무조건 차감, 반차여도 차감)
- 반차: 4시간 인정, 코어타임 중 퇴근 가능
- 휴가: 8시간 인정
- 공휴일: 8시간 인정 (주 목표에서 차감)
- 금요일: 17:00 퇴근 고정 (사용자 선호)

## 이미지 분석 방법

사진에는 다우오피스 출퇴근 기록이 표시되어 있다. 다음 정보를 추출해라:

1. **날짜**: 각 행의 날짜 (예: 2026-03-02)
2. **요일**: 월/화/수/목/금
3. **출근시간**: HH:MM 형식 ("출" 옆의 시간)
4. **퇴근시간**: HH:MM 형식 ("퇴" 옆의 시간, 당일이면 없을 수 있음)
5. **상태**: 다음 중 하나
   - `normal`: 정상 출퇴근 (출퇴근 기록 모두 있음)
   - `half`: 반차
   - `vacation`: 휴가
   - `holiday`: 공휴일/회사휴일
   - `today`: 오늘 (출근시간만 있고 퇴근시간 없음)

### 판단 기준
- 오늘 이전인데 출퇴근 기록이 아예 없는 날 = `holiday` (8시간 크레딧)
- "오늘" 표시가 있거나 출근만 있고 퇴근이 없는 당일 = `today`
- "반차", "오전반차", "오후반차" 등 표시 = `half`
- "휴가", "연차" 등 표시 = `vacation`
- "공휴일", "대체휴무", "휴일" 등 표시 = `holiday`

## 스크립트 실행 (필수)

추출한 데이터로 아래처럼 스크립트를 실행해라:

```bash
python3 scripts/calc_hours.py --data '{"days":[{"date":"2026-03-02","day_of_week":"Mon","status":"holiday"},{"date":"2026-03-03","day_of_week":"Tue","status":"holiday"},{"date":"2026-03-04","day_of_week":"Wed","status":"normal","clock_in":"10:20","clock_out":"22:46"},{"date":"2026-03-05","day_of_week":"Thu","status":"today","clock_in":"10:16"},{"date":"2026-03-06","day_of_week":"Fri","status":"normal"}]}'
```

## 출력 포맷

스크립트 실행 결과(JSON)를 받아서 아래처럼 한국어로 포맷:

```
📊 이번 주 근무현황

| 날짜 | 요일 | 출근 | 퇴근 | 실근무 | 비고 |
|------|------|------|------|--------|------|
| 03/04 | 수 | 10:20 | 22:46 | 11.43h | ✅ |
| 03/05 | 목 | 10:16 | - | 근무중 | 📍 오늘 |
| 03/06 | 금 | - | - | - | 예정 |

⏱️ 현황: 27.43h / 40h 완료 (공휴일 2일 포함) | 남은 시간: 12.57h

🎯 추천 퇴근 스케줄

옵션 1: 균등 분배
  - 목 10:16→16:50 (5.57h)
  - 금 09:00→17:00 (7h)

옵션 2: 오늘 코어타임 퇴근
  - 목 10:16→17:00 (5.73h)
  - 금 09:00→17:00 (7h)
```

## 주의사항
- **시간 계산은 반드시 스크립트를 실행해서 결과를 사용해라. 직접 계산 금지.**
- 점심시간(12:30~13:30) 차감은 스크립트가 자동 처리
- 사진이 불명확하면 사용자에게 확인 요청
- 공휴일/회사휴일 정보가 사진에 없으면 사용자에게 물어보기
