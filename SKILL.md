# Work Tracker - 다우오피스 근무시간 추적

## 핵심 규칙

**사진을 받으면 반드시 아래 순서를 따라라:**

1. 사진에서 출퇴근 데이터를 추출한다
2. JSON으로 변환한다
3. **exec 도구로 `python3 {baseDir}/scripts/calc_hours.py --data '<JSON>'`을 실행한다** ← 필수
4. 스크립트 출력 결과를 한국어로 포맷해서 응답한다

**절대로 직접 시간을 계산하지 마라. 반드시 스크립트를 실행해라.**

## 회사 규칙
- 주 40시간 근무 (월~금)
- 코어타임: 11:00 ~ 17:00
- 점심시간: 12:30 ~ 13:30 (무조건 차감, 반차여도 차감)
- 반차: 4시간 인정, 코어타임 중 퇴근 가능
- 휴가: 8시간 인정
- 공휴일: 8시간 인정 (주 목표에서 차감)
- 금요일: 17:00 퇴근 고정 (사용자 선호)

## 이미지 분석

사진에는 다우오피스 출퇴근 기록이 표시되어 있다. "출" 옆이 출근시간, "퇴" 옆이 퇴근시간이다.

추출할 정보:
- **날짜** (예: 2026-03-02)
- **요일** (Mon/Tue/Wed/Thu/Fri)
- **출근시간** (HH:MM)
- **퇴근시간** (HH:MM, 당일이면 없을 수 있음)
- **상태**: `normal`, `half`, `vacation`, `holiday`, `today`

### 상태 판단
- 오늘 이전인데 출퇴근 기록이 없는 날 → `holiday`
- "오늘" 표시 또는 출근만 있고 퇴근 없음 → `today`
- "반차" 표시 → `half`
- "휴가"/"연차" 표시 → `vacation`
- "공휴일"/"휴일"/"대체휴무" 표시 → `holiday`

## 스크립트 실행 (필수)

데이터를 추출한 후 **반드시 exec 도구로 다음 명령을 실행해라:**

```
python3 {baseDir}/scripts/calc_hours.py --data '{"days":[{"date":"2026-03-02","day_of_week":"Mon","status":"holiday"},{"date":"2026-03-03","day_of_week":"Tue","status":"holiday"},{"date":"2026-03-04","day_of_week":"Wed","status":"normal","clock_in":"10:20","clock_out":"22:46"},{"date":"2026-03-05","day_of_week":"Thu","status":"today","clock_in":"10:16"},{"date":"2026-03-06","day_of_week":"Fri","status":"normal"}]}'
```

## 응답 포맷

스크립트 실행 결과(JSON)를 한국어로 포맷:

```
📊 이번 주 근무현황

| 날짜 | 요일 | 출근 | 퇴근 | 실근무 | 비고 |
|------|------|------|------|--------|------|
| 03/02 | 월 | - | - | - | 🔴 공휴일 |
| 03/03 | 화 | - | - | - | 🔴 공휴일 |
| 03/04 | 수 | 10:20 | 22:46 | 11.43h | ✅ |
| 03/05 | 목 | 10:16 | - | 근무중 | 📍 오늘 |
| 03/06 | 금 | - | - | - | 예정 |

⏱️ 현황: 27.43h / 40h 완료 (공휴일 2일 포함) | 남은 시간: 12.57h

🎯 추천 퇴근 스케줄

옵션 1: 균등 분배
  - 목 10:16→16:50 (5.57h)
  - 금 09:00→17:00 (7h)

옵션 2: 오늘 코어타임 퇴근
  - 목 10:16→17:00 (5.73h)
  - 금 09:00→17:00 (7h)
```

## 주의사항
- **시간 계산은 반드시 exec 도구로 스크립트를 실행해서 결과를 사용해라**
- 사진이 불명확하면 사용자에게 확인 요청
- 공휴일 정보가 사진에 없으면 사용자에게 물어보기
